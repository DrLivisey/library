Документация к библиотекам.

SHT21.h
Датчик не нуждается инициализации и общаться с ним можно только с помощью команд.

float SHT21_get_Rh (); возвращает - данные обработанные данные о влажности в % в целых числах чаще всего, но на всякий случай с плавующей точкой

float SHT21_get_Temperaturу (); вовзращает обработанную температуру в градуах цельсия с плавающей точкой

uint8_t SHT21_get_UR (); возвращает даанные регистра юзера, там особо настроек нет, кроме битности данных. По умолчанию они настроены на максимальную длинну и я не стал ее менять.

SHT21_restart (); перезапуск датчика


BMP180.h
все функции кроме получения данных давления в 16 битном формате, давление в 32 битном, просто число не влезает в 16 бит. Все они со знаком.

int16_t BMP180_init(); Инициализация, во время которой принимаются калибровочные значения. Без них датчик будет выдавать не то.

int16_t BMP180_get_UP(); возращает сырые данные давления без обработки.

int16_t BMP180_get_UT ();  возращает сырые данные температуры без обработки.

int16_t BMP180_get_Temperature (); возвращет обработанные данные температуры, с очень странными вычислениями. Не факт что их обязательно делать на МК, поэтому и есть две верхние функции

uint32_t BMP180_get_Press (); возвращает обработанные данные давления. Вычисления еще хуже чем в температуре.


CCS811.h (CJMCU811)

CCS811_init(); инициализация, перевод датчика в рабочее состояние

CCS811_Change_Mode (uint8_t mod); изменение режима его работы. Вместо mod просто десятичное число от 0 до 4.

0: Mode 0 – ожидание. По умолчанию в этом состоянии, чтобы датчик начал измерения - необходимо перевести в любой другой
1: Mode 1 – обычный режим - измерения раз в секунду
2: Mode 2 – измерения раз в 10 секунд
3: Mode 3 – измерения раз в минуту
4: Mode 4 – измеренияраз в 250 мс.
Остальные режими заблочены.Любой режим кроме представленных - перевод в ожидание (0 режим)

uint16_t CCS811_Get_eCO2 (); возвращает 16 битное значеие eCO2

uint16_t CCS811_Get_TVOC (); возвращает 16 битное значеие TVOC

CCS811_Add_Env_data (uint8_t humidity, uint8_t temperature); изменение калибровочных регистров, по умолчанию они 50% влажность и 25 С. Сначала влажность, потом температура.

void CCS8811_Restart (void); перезапуск датчика
